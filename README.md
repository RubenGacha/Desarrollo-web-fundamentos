# Desarrollo web

## ¿Que es la programacion? 

La programacion puede definirse como un arte ya que depende de la creatividad del programador algo que todas las personas tenemos para resolver un problema, necesidad o porque queremos hacer algo diferente.

lo importante es entender tanto contexto como el objetivo de lo que se quiere hacer, esto nos permite generar hipotesis o supuestos que iremos validando las consecuencias de realizarlo de esa manero o de otra.

## Entender el QUÉ para después ejecutar el CÓMO

Muchas personas se enfocan inmediatamente en el "cómo":

+ "¿Qué herramienta uso?"
+ "¿Qué procedimiento sigo?"
+ "¿Qué tengo que hacer?"

Pero antes conviene preguntarse:

+ "¿Qué estamos intentando lograr exactamente?"
+ "¿Cuál es el problema real?"
+ "¿Qué resultado buscamos?"

### Ejemplo práctico

Imagina que tu jefe dice:

> "Necesitamos más ventas."

**Pensamiento reactivo (centrado en el cómo):**

* Lanzar descuentos.
* Hacer más publicidad.
* Llamar a más clientes.

**Pensamiento proactivo (empezando por el qué):**

* ¿Qué significa exactamente "más ventas"?
* ¿Queremos más clientes o que los actuales compren más?
* ¿El problema es la captación, la conversión o la retención?

Una vez entendido el **qué**, eliges mejor el **cómo**.

### Resumen en una frase

La idea podría reformularse así:

> **Antes de actuar, comprende el objetivo y el contexto; luego decide la mejor forma de ejecutarlo, en lugar de responder automáticamente a los acontecimientos o instrucciones.**
