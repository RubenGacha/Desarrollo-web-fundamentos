# ¿Que es HTML?

+ Es un lenguaje de marcado de hipertexto (Hyper Text Markup Languaje)
+ HTML no es un lenguaje de programación, es un lenguaje de estructura.
+ Es la base con la que están creadas **TODAS** las páginas web del mundo.
+ Cada etiqueta le dice al navegador y a los motores de búsqueda cuál es la estructura de los documentos, elementos, organizacion, etc. 

## Historia

+ 1989 - Inicio del proyecto World Wide Web por Tim Berners-Lee en CERN.
+ 1991 - Lanzamiento de la World Wide Web al público (basada en HTTP, HTML y URL).
+ 1991 - Primera especificación de HTML publicada por Tim Berners-Lee.
+ 1994 - Creación de la W3C (World Wide Web Consortium), organismo encargado de los estándares web.
+ 1995 - HTML 2.0 (primer estándar oficial de HTML).
+ 1997 - HTML 3.2 (primera recomendación HTML de la W3C).
+ 1997 - HTML 4.0 (introduce separación entre contenido y presentación mediante CSS).
+ 1999 - HTML 4.01 (última revisión de HTML 4 y una de las versiones más utilizadas).
+ 2000 - XHTML 1.0 (reformulación de HTML 4.01 basada en XML).
+ 2004 - Creación de la WHATWG (Web Hypertext Application Technology Working Group).
+ 2008 - Primer borrador público de HTML5 impulsado por WHATWG y posteriormente apoyado por W3C.
+ 2014 - HTML5 se convierte en recomendación oficial de la W3C.
+ 2016 - HTML 5.1.
+ 2017 - HTML 5.2.
+ 2019 - WHATWG se convierte en la autoridad principal del estándar HTML; W3C y WHATWG acuerdan un único estándar.
+ 2020 - HTML pasa a mantenerse como un "Living Standard" (estándar vivo), actualizado continuamente por WHATWG.
+ 2026 - HTML continúa evolucionando como Living Standard, incorporando nuevas APIs y características web sin depender de versiones numeradas.
